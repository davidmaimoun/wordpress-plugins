# SmartBiDi – WooCommerce Product BiDi Fix

## Description

**SmartBiDi** automatically detects mixed RTL (Hebrew, Arabic, etc.) and LTR (Latin) text in **WooCommerce product names and descriptions** and applies proper bidirectional formatting (`dir="rtl"` / `dir="auto"`).

This fixes display issues for products with mixed Hebrew/Latin or Arabic/Latin text.

---

## Features

- Automatic BiDi detection for WooCommerce product titles and descriptions  
- Applies `dir="rtl"` for RTL-only text  
- Applies `dir="auto"` for mixed text  
- Lightweight and fast  

---

## Installation

1. Upload the `smartbidi` folder to `/wp-content/plugins/`  
2. Activate the plugin from the **Plugins** menu  

---

## Usage

- Works automatically for WooCommerce products — no configuration needed  
- Ensures correct display of mixed RTL + LTR product names
