== Description ==
SmartBiDi detects mixed RTL (Hebrew, Arabic, etc.) and Latin text in WooCommerce products. 
It allows you to force RTL display on Navbar, Content, and Footer separately, ensuring proper layout for RTL languages.

== Installation ==
1. Upload the `smartbidi` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the plugin settings and configure which zones should display RTL.

== Frequently Asked Questions ==
= Does this plugin work with all themes? =
It works with most themes that follow standard WordPress and WooCommerce conventions. Custom themes may require additional CSS adjustments.

= Can I force RTL only for certain products? =
Yes, you can control the RTL zones per product using the plugin settings.
